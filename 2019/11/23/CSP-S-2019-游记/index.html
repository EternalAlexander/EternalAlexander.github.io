CSP-S 2019 游记

发表于 2019-11-23 更新于 2019-12-18 阅读次数：

一年来毫无长进啊。

Day 0

​ 上午整个机房都在颓。我在打以撒羔羊版，感觉羔羊确实比重生难很多。然后梅老板突然来了，辛亏是从前门进来的，我快速关掉了游戏，切到python IDLE然后开始写我也不知道是什么的东西。然后被梅老板拍了几下，说了几句加油的话。实在是tkpl。

​ 然后吃完了饭就和ustze一起坐松的车去试机了。远远地看到了蒋队，然后见到了xht37，拿到了我和ouuan的神秘奖品。有些伤感地发现已经没有多少认识的选手了，去年HB省队现在已经没剩几个人了。

​ 写了个LCT然后一交90，TLE了。调了一会然后弃疗，粘了之前的代码，发现cut写挂了…断不存在的边会死循环。

​ 然后听了下注意事项之类的，就走了。晚上打了会空洞骑士45，见到辐光然后一阶段暴毙。

Day1

​ 和去年和前年差不多地进了考场，不过这次是在1楼。写了个A+B试了试机，然后就等密码了。

​ 看题，T1第一眼没看懂，先跳了，T2好像很可做，T3完全没想法。想了想决定还是顺开。仔细看了下T1，发现好像是个模拟，就直接写了，测了下大样例过了就没管了。T2稍微想了两分钟怎么实现比较容易，然后很快写完了。一开始有些细节想错了调了一会。过了大样例之后我开了一下大样例，发现是114514个点的一条链，看上去很不靠谱的样子，就又花了接近十分钟写了个对拍拍了一下，然后发现fc不能用？？？搞了下发现可以用diff，然后拍上了。

​ 这时候还有接近三个小时给我搞T3，感觉又是去年那种随便AK的Day1，然而被这个T3教育了。我先想了半个小时，发现我不会做，然后觉得先考虑下链怎么做再推广，然后又想了半个小时发现这个我也不会，然后想考虑下菊花图推广出来，然后发现这个我也不会。最后不到一个小时的时候想到一个可能对的树上的 $\mathcal O(n^4)$ 的做法，直接开码，然后还有10分钟的时候测样例发现WA了，输出了类似 0 0 0 0 5 这种东西。然后就弃疗了，检查了下文件，感觉要被大众分吊打90分。

​ 然后出来一问发现都 210，就不是很慌了。但突然听说T1要ull，我完全没注意，可能挂成50分。这下完了，随便来个小朋友都能踩我了。

​ 晚上心情很不好，打了会 Celeste 8A，然后过不去，自闭了。

Day 2

​ 昨天被大众分吊打50分，今天得多写50分，压力有点大但莫名非常地兴奋。

​ 到了座位发现左边是zar，再左边是另一个外初的女生。突然莫名地怀念起外初来了。然后开题，T1怎么有 $998244353$ ？跳了。T2 长得很斜率优化，先跳了。 T3 一脸可做但好像很难写，我前几天看toptree的时候好像看到过什么non-local search，link cut维护重心之类的，不过最后再说吧。

​ 发现没有能秒的题，开始有点慌了。我怕最后摇摆不定什么都没写出来，于是决定强迫自己顺开。T1仔细看了下发现超过一半的最多一种，然后就随便dp一下。写了发现居然是 $\mathcal O(n^3m)$ 的，然后又想了五分钟发现可以改下状态弄成 $\mathcal O(n^2m)$ 。大概 30+min就过大样例了。

​ T2 依然没思路，就先写了个 $n^3$ dp，然后我感觉dp值有单调性，打了个表果然是的，然后就可以 $n^2$ 了。然后我猜决策也有单调性，打了个表果然是的，但这个式子好像只能cdq分治之类的苟到 $nlogn$ ，一脸过不去的样子。没管这么多先把高精度写了，但没想到写了很久很久，最后还写挂了。调出来大概用了40min。然后想了想决定写个树状数组的 $nlogn$ ，说不定能跑呢，然后写了十几分钟过了样例，然后开始写题面那个数据生成器，对着它盯了好几分钟才明白它是在干什么，然后终于调过那个大样例的时候已经只剩1h了，只可惜1e7都要跑2s，我又卡了一会常数，想了想能不能线性，最后感觉没啥希望，决定弃了。突然发现我MLE了，想了想反正也跑不过去开 8e6 算了。

​ 这时候还有30min左右，我觉得写不完T3了，但75好像很能写，不知道写不写得完。直接开始rush，暴力重心写完过了10min，然后写链发现过不去样例，然后发现不一定是顺序一条链，还得写个dfs编号之类的，写完只有不到10min了，我很想继续rush这个二叉树，但理智告诉我不能写下去了，我把无关文件都删了，然后每个题测了一遍大样例，放虚拟机测了下有没有CE，签了字，最后一分钟啥也没干了。

​ 出来感觉还行，但T3又是所有人 75 ~ 100，就我 55。

​ 下午过掉了8A，然后回学校了。

​ 估分 $(50\sim95)+100+10+100+88+55=403\sim448$ ，被去年的自己吊着打。现在只能希望T1出题人仁慈一点吧。

Day 3+

​ 测了下全省程序，假如T1不挂多少分好像还行的样子。蒋队535，表示只能orz了。

​ 稍微想了下d2t3感觉非常简单的样子，早知道就不写垃圾高精度了。

​ 学了会文化课发现好难好难好难好难好难，自闭了。

不知道Day几了

​ 出分了，$432$，d1t1 $95$ ，d2t1后面不知何故全T了只有 $84$。非常生气，交了50块钱申诉。

好久以后了

​ 申诉成功了，现在 $448$ 了。

​
